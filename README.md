#  PLC Traffic Light Control System

An industrial automation project developed using Siemens PLC and WinCC HMI to simulate an automated traffic signal control system. The project uses ladder logic programming and timer-based sequencing to control Red, Yellow, and Green traffic lights while providing real-time monitoring through an HMI interface.

## Key Features

* Automatic traffic light operation
* Timer-based signal sequencing
* PLC Ladder Logic Programming
* WinCC HMI Visualization
* Real-time signal status monitoring
* Simulation in Siemens TIA Portal

## Technologies Used

* Siemens TIA Portal V17
* Siemens S7-1200 PLC
* WinCC Runtime Advanced
* Ladder Logic (LAD)

## Project Objective

To design and simulate a reliable traffic light control system using PLC programming concepts, demonstrating the application of timers, sequencing logic, and HMI integration in industrial automation.
